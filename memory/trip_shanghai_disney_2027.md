- Обратно: 3 апреля, Шанхай → SVO, вылет 10:00, прилёт ~17:00
- SVO → Уфа: нужно добавить рейс обратно после прилёта 3 апреля ~17:00

**Билеты:** ~800 000 руб (семья с детьми) + ~40 000 руб УФА→МСК
