После завершения любой задачи, которая изменяет файлы в проекте — коммитить и пушить в репозиторий без запроса подтверждения.

**Why:** Пользователь не хочет повторять эту инструкцию в каждом чате.

**How to apply:** По завершении задачи (или логического блока работы): `git add` нужные файлы → `git commit` → `git push`. Не спрашивать "делать ли коммит?" — делать.
