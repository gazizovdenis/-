# Хранилище

Мастер-индекс всех подпапок в Хранилище. Здесь собраны материалы по проектам и личной сфере.

## Подпапки

| Префикс | Название | Описание |
|---------|----------|----------|
| 00_ | Мусорка | "Удалённые" файлы через mv по разрешению. Журнал в _log.md |
| 01_ | Маркетплейс | Продажа солнцезащитных очков на ВБ и Озон, тестирование новых категорий, выход на Озон глобал |
| 02_ | Технопродукт | Продажа б2б запчастей к оборудованию линии розлива + товары под санкциями |
| 03_ | Личные_проекты | Автоматизации собственных операционных процессов |
| 08_ | Личное | Личные данные: здоровье, обучение, хобби, распределение |
| 95_ | Текущие_задачи | Активные задачи, не привязанные к конкретному проекту |
| 99_ | Остальное | Буфер несортированного материала |

## Правила работы
- Кладите материалы как есть, без фильтрации.
- Для каждого проекта — README.md и Заметки_чат.md.
- Новые материалы → 00_Inbox/ в соответствующей папке, потом сортировать.

## Новая структура
Сейчас все проектные папки находятся внутри `raw/`. `wiki/` содержит обзорные страницы и карту навигации.
- `raw/` — сырой источник, оригинальные файлы.
- `wiki/` — обзорные страницы, которые ИИ читает первыми.
- `raw/README.md` объясняет структуру raw.
- Старые корневые копии перемещены в `_backup_root_copies_20260515/`.
