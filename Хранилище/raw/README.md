# raw

**Summary:** Сырые материалы по проектам и личной сфере. Здесь лежат оригиналы, которые не редактируются после загрузки.

## Структура raw/

- `00_Мусорка/` — удалённые файлы и переносы.
- `01_Маркетплейс/` — материалы проекта Маркетплейс.
- `02_Технопродукт/` — материалы проекта Технопродукт.
- `03_Личные_проекты/` — материалы по автоматизациям и личным проектам.
- `04_Все_звонки_и_встречи/` — заметки и транскрипты всех звонков.
- `08_Личное/` — личные записи, здоровье, обучение, хобби.
- `95_Текущие_задачи/` — задачи, которые решаются сейчас.
- `99_Остальное/` — буфер несортированных материалов.

## Принципы работы

- `raw/` содержит только оригиналы.
- Файлы из raw не редактируются напрямую.
- Для обзора и принятия решений используем `wiki/`.
- Если материал нужно сохранить как обсуждение или вывод, создайте отдельную wiki-страницу.
