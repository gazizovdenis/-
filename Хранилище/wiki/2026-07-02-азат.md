## Новый регламент (от Ивана)

- Утром — план на день в чат.

- Минимум **20 новинок/месяц** на поиск.

Денис объяснил логику: регламент не для жёсткого контроля, а для ощущения структуры и безопасности. Результаты важны, не часы.

## Анализ сотрудника

### Что нравится
- **Свобода** (нет жёстких регламентов "каждые полчаса") — называет первым. Считает себя эффективнее без жёсткого расписания.
- **Скорость процессов**: нашёл товар → сразу заказ → сразу запуск. У предыдущего: 2–3 недели на согласование с директором.
- **Развитие**: не стоит на месте, уже продвигает.

### Что не нравится
- Пока **ничего**. Прямой ответ, без натяжки.

### Идеи по улучшению
- Предложений по системе нет. Есть опасение — незнакомый менеджер может делать рекламу неправильно (видел: по поиску рекламы без отзывов — это ошибка). Хочет знать, кто будет масштабировать.
- **Сигнал:** нет инициативных идей по улучшению. Скорее исполнитель, не архитектор.

### Сигналы вовлечённости
- Высокая энергия, амбициозен (мечта — 200к, Москва, рост).
- Дофамин от результата — когда товар залетает. Несёт ответственность за процесс.
- Тревога за качество менеджера = болеет за результат своего поиска.
- **Не против регламента** после объяснения логики — гибкость, зрелость.

## Схема работы (обновлённая)

| Элемент | Детали |
|---------|--------|
| Фикс | 30к сейчас (возможно временное повышение для поддержки перехода) |
| Фокус | Только поиск новинок — не масштабирование |
| Минимум новинок | 20/мес |
| Премия | Градация по выручке новинки: 500к → 1М → 3М+ → другой размер; ориентир 10–100к за залетевшую новинку |
| Без штрафов | Если новинка не зашла — ничего не теряет |
| Связка | Нашёл → первично завёл → передал менеджеру (Дмитрий или Анна) |
| Площадки | ВБ + Озон (уточнить по Ozon отдельно) |

**Цель:** через 2 мес (сент 2026) Азат полностью уходит с другой работы → к Денису full-time.  
Денис готов временно поднять фикс для поддержки этого перехода, потом перейти на КПА.
