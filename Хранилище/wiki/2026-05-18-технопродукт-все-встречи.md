## Часть 3 — Итоговая встреча со Светланой

**Главный вывод:** Кризиса бизнес-модели нет. Есть сильная экспертиза и вовлечённая команда. Проблема — хаотичное управление. Пока нет системы, масштабирование только усилит хаос.

### Ключевые выводы

**1. Приоритет на 2 месяца: систематизация, не новые гипотезы**
- Команда в хаосе не примет новые задачи — будет саботаж
- Матрица Ансофф: сначала **существующий рынок + существующий продукт**, потом новый продукт, потом новый рынок

**2. Нет продаж — есть операторы**
- Никто, кроме собственников, не продаёт текущую услугу — это потеря денег
- Нужен **аккаунт-менеджер** (2 KPI: развитие текущих клиентов + поиск новых)

**3. Критический риск: зависимость от одного клиента**
- Эфес = **50–60% всей выручки** — недопустимая концентрация
- Гипотезы на Сибур → ещё больше растят зависимость

**4. Позиционирование отсутствует**
- 4 сотрудника дали 4 разных ответа на вопрос «чем занимается компания»
- Менеджер не может продавать без чёткого оффера

**5. Найм: сначала операционный директор, потом продажник**
- Операционщик снимает 80% хаоса и сам найдёт продажника
- Искать **по навыкам** (управление + продажи), не по экспертизе (технари не продают)
- Целевой профиль: ~Мария (сотрудник Владимира, ~70% совпадения)
- Зарплата: 120–150k + %

**6. Метрики сотрудников**
- Главная метрика + опережающие показатели на каждую должность
- Азамат: роль «универсального солдата» — не понимает свою метрику, нужна оцифровка

**7. Мотивация через выгоду, не через приказ**
- Задачи сверху = рикошет. Продавать идею через личную выгоду сотрудника

**8. Партнёрство Денис + Руслан**
- Энергия/видение (Руслан) + чёткость/структура (Денис) — уникальное сочетание
- Руслан тратит энергию на операционку вместо развития — нужно исправить
