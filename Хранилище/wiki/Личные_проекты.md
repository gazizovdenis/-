## Автоматизации (текущее)

- **scripts/transcribe.py** — универсальный скрипт транскрибации (faster-whisper small + OpenAI). Поддерживает `--engine local|openai`, пропускает уже существующие транскрипты. Используется для всех встреч и сессий.
- **Диаризация** — настроена через pyannote.audio (HuggingFace токен). Скрипт: skills/transcribe-media/.
- **carousel_ai.html** — генератор карусельных постов для Instagram через AI (локальный инструмент).

## Instagram / Контент

Стиль постов проанализирован по 1222 постам. Паттерны сохранены в memory (instagram_style.md):
- Личный, тёплый, честный тон — без «успешного успеха»
- Структура: зацепка → личная история → вывод → вопрос
- Первое лицо, скобки для самоиронии, умеренные эмодзи

**Планируется:** контент-завод через Hitxel ($15/мес, AI-видео) — ветка от Ивана с X Forum.
