## С чего начать новый запрос

```
Что хочу сделать?
│
├── Узнать что-то обо мне или моём профиле
│   └── → wiki/Портрет.md
│
├── Работа по проекту
│   ├── Маркетплейс (ВБ, Озон, очки, категории)
│   │   └── → wiki/Маркетплейс.md → raw/01_Маркетплейс/
│   ├── Технопродукт (запчасти, санкции, б2б)
│   │   └── → wiki/Технопродукт.md → raw/02_Технопродукт/
│   └── Личные проекты (автоматизации)
│       └── → wiki/Личные_проекты.md → raw/03_Личные_проекты/
│
├── Звонки, встречи, транскрипты
│   └── → wiki/Все_звонки_и_встречи.md → raw/04_Все_звонки_и_встречи/
│
├── Текущие задачи / дедлайны
│   └── → raw/95_Текущие_задачи/README.md
│
├── Личное (здоровье, личные дела)
│   └── → raw/08_Личное/README.md
│
├── Новый материал — куда положить?
│   └── → raw/<нужный_проект>/00_Inbox/ или Встречи/ или Документы/
│       → после обработки: попросить обновить wiki/
│
└── Не знаю с чего начать
    └── → wiki/index.md (карта всех страниц)
```

---

## Правила работы с системой

| Ситуация | Действие |
|----------|----------|
| Новый файл | Положить в `raw/`, назвать `YYYY-MM-DD_имя.md` |
| Удалить файл | `mv` в `Хранилище/raw/00_Мусорка/`, записать в `_log.md` |
| Обновить знания ИИ | Попросить обработать `raw/` → обновить нужную `wiki/` страницу |
| Не знаю куда | `raw/<проект>/00_Inbox/` — разберём позже |

## Иерархия источников истины

```
CLAUDE.md          ← главный контекст (читается при каждом сеансе)
    ↓
wiki/*.md          ← обработанные выжимки, актуальная картина
    ↓
raw/**/*.md        ← первичные материалы, источник для wiki
```
