# product-analysis

Скилл для анализа новых товарных категорий для маркетплейсов ВБ и Озон.

## Триггеры
- "найди новую нишу"
- "проанализируй категорию"
- "подбери товар для маркетплейса"
- "что можно зайти на ВБ"
- "анализ новинки"
- "новый товар для маркетплейса"

## Контекст бизнеса
- Уже работаем в нише «солнцезащитные очки» (основной сезон: март–июль)
- Задача: найти категорию с выручкой в период август–февраль или равномерно весь год
- Закупка через Китай (прямой импорт, FOB)
- Новое юридическое лицо

## Фильтры отбора категорий
Все критерии желательно выполнять одновременно:

| Параметр | Норма | Минимум |
|----------|-------|---------|
| Процент выкупа | ≥ 80% | ≥ 60% |
| Чистая маржа | ≥ 25% | ≥ 20% |
| Ёмкость (выручка топов на 1 SKU) | ≥ 1 млн руб./мес | — |
| Логистика | некрупный, нехрупкий, без спецхранения | — |
| Источник | прямая закупка в Китае | — |

Дополнительный плюс: возможность масштабироваться до 450 млн руб./год в рамках одного юрлица.

**Не рассматривать:** товары на падающем тренде.

## Алгоритм работы

1. Подобрать минимум 5 категорий-кандидатов под заданные фильтры
2. По каждой категории собрать данные по 8 пунктам (см. ниже)
3. Вывести таблицу + комментарий по каждой нише
4. Дать итоговую рекомендацию: топ-2 для входа

## Что анализировать по каждой категории

1. Название ниши (категория ВБ/Озон)
2. Оценочный % выкупа
3. Комиссия ВБ и Озон (%)
4. Средний чек и средняя закупочная цена (FOB Китай)
5. Юнит-экономика: выручка → МП-расходы → маржа → чистая прибыль
6. Сезонность (индекс по месяцам или текстово)
7. Конкуренция: кол-во продавцов, кол-во карточек, кол-во карточек с движением
8. Риски (конкуренция, регуляторика, специфика)

## Оценка каждой категории
- **Зелёный** — подходит, можно заходить
- **Жёлтый** — условно подходит, дать рекомендацию как заходить в конкуренцию
- **Красный** — не подходит + почему

## Формат результата
Таблица по всем кандидатам + развёрнутый комментарий по каждой нише.
Приоритет — осенне-зимние и внесезонные товары.
Подсвечивать тренды и фрагментированность рынка.

## Правила
- Товары на падающем тренде не рассматриваем
- Если категория жёлтая/зелёная — обязательно дать рекомендацию по стратегии входа
- Учитывать конкуренцию: кол-во продавцов, карточек и карточек с движением
